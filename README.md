# TelaPrime

TelaPrime é um projeto web desenvolvido em **HTML** e **CSS** que simula uma vitrine profissional
de TVs e monitores.

O objetivo do projeto é apresentar uma página comercial responsiva, com catálogo de produtos,
imagens dos anúncios, preços, condições de venda, links para ofertas e formulário de atendimento.

O site foi criado para aplicar conceitos de desenvolvimento web como estrutura semântica,
acessibilidade, layout responsivo, Flexbox, Grid, mobile first, medidas relativas e animações CSS.

## Tecnologias

- HTML
- CSS

## Como abrir

Abra o arquivo `index.html` em um navegador.
