# TelaPrime

Projeto acadêmico em HTML e CSS simulando uma loja profissional de TVs e monitores. O site exibe
6 televisores e 3 monitores com imagem do anúncio, preço, parcelamento, vendedor/fonte e link de
compra.

## Como abrir

Abra `index.html` no navegador.

## Apresentação

O arquivo de PowerPoint é `apresentacao-telaprime.pptx`.

## Onde cada conceito aparece

- HTML semântico: `header`, `nav`, `main`, `section`, `article`, `aside`, listas, formulário e
  `footer`.
- Acessibilidade: link para pular ao conteúdo, `aria-label`, `alt` nas imagens, campos com
  `label`, controles nativos, contraste e foco visível com `:focus-visible`.
- Flexbox: menu, botões, especificações dos produtos, opções do formulário e rodapé.
- Grid: hero, vitrine de TVs, vitrine de monitores, condições comerciais e atendimento.
- Mobile first: layout inicial em uma coluna e media queries com `min-width`.
- Medidas relativas: `rem`, `%`, `fr`, `min()`, `aspect-ratio` e `svh`.
- Responsividade: adaptação de cards, hero e comparativos para telas pequenas e grandes.
- Animações: transições de hover e `@keyframes pulse`, respeitando `prefers-reduced-motion`.

## Roteiro sugerido

1. Apresentar a proposta da loja TelaPrime.
2. Demonstrar os cards com imagem do anúncio, preço, parcelamento, vendedor e fonte.
3. Demonstrar a página no celular e no desktop.
4. Mostrar a estrutura semântica do HTML.
5. Demonstrar navegação por teclado e formulário acessível.
6. Explicar no CSS onde estão Flexbox, Grid, media queries, medidas relativas e animações.
